Check out some of my favorite projects from my time at Ada:



task list API

Technologies: Python

Skills: Relational databases (PostgreSQL), External APIs (Slack), App deployment

Description: This project is a RESTful API that allows the user to create tasks associated with goals. When tasks are completed, a Slackbot posts a message to a Slack workspace.


personal portfolio website

Technologies: HTML, CSS

Skills: Semantic HTML, CSS Styling

Description: This is my very first static HTML/CSS project! ♥


weather report

Technologies: JavaScript, HTML, CSS

Skills: Front end web development, external API use (OpenWeather & LocationIQ), pair programming, promises

Description: Weather Report was made in collaboration with Junnie Lee, an Ada Developers Academy colleague. It is a front end website that accepts a location from the user and returns realtime temperature data.


inspiration board

Technologies: ReactJS, Python, React Router

Skills: Full stack web app development, teamwork, relational databases

Description: Inspiration Board was made in collaboration with Ada Developers Academy Colleagues: Morgan Heuer, Emily Saeli, and Sarah Williams. It is a full stack web app that allows user to post, like, and delete messages to virtual message boards.


what the fridge

Technologies: Google Firebase, ReactJS, React Router

Skills: noSQL database (Google Cloud Firestore), full stack web app development

Description: What the Fridge is a full stack web app made in three weeks as my capstone for Ada Developers Academy. It has two main features: kitchen inventory management, and recipe finding. Users can add ingredients from their kitchen, update the quantities, and delete ingredients, then find recipes based on those ingredients.
