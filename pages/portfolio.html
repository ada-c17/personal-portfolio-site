Ada coding journey

I start my journey at Ada in this March 2022. I have learning a lot from this program including soft skills and coding skills. Regarding coding skills, I learn about Python, back-end development such as Web server, Flask, and database, and recently I just start to learn about Front-end development like HTML and CSS. Honestly, I feel like I am in progress being software developer right now. In addition, I really enjoy practicing PSE and industry prep every week. Meantimes, Ada also assign student some project for each unit to cover what students had learned. Until now, I had completed some Ada projects such as:
  • Viewing Party
  • Adagrams
  • Swap Meet
  • Task List API

Viewing Party

This is Ada first project that cover the learn about algorithm of conditional logic, lists, Dictionaries, nested loops, and nested data structures. From this project, I learned how to access value in the nested data structures, to refactor my code, to create helper function for reusable code.

Adagrams

This project practice pair programming technique. It is a good project because I got a chance to know my partner, to learn from each other, to cooperate and work with each other to reach the goal, and to practice using git to maintain code.

Swap Meet

Swap Meet project is covered the OOP concept. It gave me the idea on creating classes, importing modules, overriding methods from superclasses and object, and implementing inheritance and composition.

Task List API

Task list api is project that implement the back-end development with flask and database, which cover the concept of creating models, route, RESTful CRUD, environment variable and one to many relationship between two table, and making request and response. This project took longer than other project to complete for me. Although, I really enjoy working on this project because I learned to create app by using Flask and to deploy it into the internet.

Conclusion

I can see my improvement of coding after working on these projects as I experienced different solving problems, different difficult levels, and I could accomplish it. To be honest, It was hard at beginning of every project, but it got smoother after I got familiar it. Meantimes, I always go back to learning lessons, googling, and go to office hour to get extra help when I really get stuck.
