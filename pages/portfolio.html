Task List API

A Web API using Python and Flask created to organize tasks and goals. Tasks can be created, deleted, sorted, marked complete, and added to goals. Integrating the Slack API, a message is sent via Slack to notify when a task is marked "complete".

Skills Assessed:

  • Demonstrating understanding of the client-server model, request-response cycle and conventional RESTful routes
  • Reading and using existing external web APIs
  • Creating a one-to-many relationship between two models
