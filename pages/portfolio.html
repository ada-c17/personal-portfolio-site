Adagrams

This Project worked similarly to classic word games like scrabble, or bananagrams. First the code generates 10 random letters, each with different point values. Then it's up to the user to try to come up with a word using available letters, to get the highest score possible! This project had a lot of focus on getting comfy with dictionaries and iteration.

Task List API

This project involved developing, and deploying a Web API that keeps track of tasks and goals. Some cool features include: Assigning smaller tasks to larger goals, marking a task as complete, and when a task is completed a message is automatically posted on a slack channel letting others know it's taken care of. This project utilized Python, Postrgress, SQLAlchemy, Postman, Heroku, and Flask! as you can see this is further in the ada program and we're learning how to utilize more languages and applications!
