Viewing-Party

Love watching shows & movies? Getting tired of keeping track of all the movies you and your friends like/dislike/have watched/want to watch etc.? Adagrams

Who likes playing Scrabble? If you're not familiar with Scrabble, it's a game where players draws from a pool of tiled letters to create a word. Adagrams is a type of anagram similar to Scrabble. Check out this project and implement this Python code and see how many words you can come up with!

Swap-Meet

Have you ever gone to an actual swap-meet? It's truly an exciting feeling! What's more exciting is creating it all from your desktop! Check out this project to get a feel for a virtual meetup and keep track of various inventories all through code! It's great for those looking to explore more into OOP.

Task-List

Keeping To-Do lists can help us keep track of what needs to be done in our busy lives! It's defintely not effective to write it down on paper or worse, make a mental note of it in your head! Why not create a task-list using Python to keep track of Tasks and Goals that you need to complete? Check out this project that uses Python, SQL, Flask, and APIs
