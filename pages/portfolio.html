Ada FUN-damentals

Viewing Party

In Viewing Party, bright-eyed Ada students solve the age-old question: How do we all agree on a movie to watch? Fresh off an introduction to Python, they use their newfound knowledge of conditional logic, lists, dictionaries, and nested loops and data structures to implement a series of movie-themed functions.

Adagrams

Word games are all the rage in 2022, and Ada students board the hype train with Adagrams. Using the power of friendship (i.e. pair programming) and test-driven development, they implement a series of Python functions in support of the anagram-based Adagrams puzzle game.

Project Repo

Swap Meet

Ada students organize the exchange of goods in their communities with Swap Meet. Object-oriented programming is the skill of the hour, and they use classes and instance methods to describe and modify Vendors and the Items in their inventories.

Back-end

Task List API

It sometimes seems like there's never enough hours in the day, so Adies get organized with Task List API. Storing their to-dos in a postgres database, they write RESTful CRUD routes to sort and organize that pesky task backlog. To keep up morale, Task List calls the Slack API to post congratulations upon task completion.
