Task List

A RESTful API deployed on Heroku. Written with Python, Flask, & SQLAlchemy.

Task List manages a user's tasks and goals list using CRUD principals. When a task is marked completed, a Slack Bot posts a little message in my private server.

Viewing Party

OOP program written in Python. Developed with TDD.

Viewing Party is a program that keeps tracks of movies you and your friends have watched or want to watch. The program can give suggestions for movie night based on: hosting availability, rating, genre, who's watched it, etc. This was my first dive into nested data structures.

Swap Meet

OOP program written in Python.

Swap Meet keeps track of user's inventory & helps calculate item swaps based on availability, condition, age, category, etc.

Solar System

Paired project. A RESTful API written with Python, Flask, & SQLalchemy.

Using CRUD methods, Solar System allows a user to maintain a database of planets - either real or fictional!
